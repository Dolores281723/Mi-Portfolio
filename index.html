Sobre mí

Apasionada por la tecnología y la resolución de problemas, mi viaje en el mundo del desarrollo comenzó explorando los cimientos de los sistemas informáticos. Esta curiosidad me llevó a la programación orientada a objetos con Java, donde aprendí a construir software robusto y escalable. Ahora, he volcado toda esa energía en el desarrollo web. Con HTML, CSS y JavaScript, disfruto transformando la lógica en interfaces atractivas y fáciles de usar. Cada proyecto es una nueva oportunidad para aprender y superar retos. Estoy entusiasmada por seguir creciendo y colaborar en la creación de tecnología con impacto.


"Actualmente busco oportunidades como desarrolladora Front-End donde pueda aplicar mis habilidades y seguir creciendo en un equipo dinámico."

Mis Proyectos

Proyecto Ventanas

Página web corporativa para una empresa de ventanas. El reto principal fue crear una galería de productos interactiva usando JavaScript puro para filtrar y mostrar información sin recargar la página. Se implementó un diseño 'mobile-first' con CSS Grid y Flexbox para asegurar una experiencia óptima en todos los dispositivos.

Ver Demo Código en GitHub

Proyecto Restaurante

Web para un restaurante moderno, enfocada en presentar el menú y la atmósfera del local de forma visualmente atractiva. Desarrollada con HTML, CSS y JavaScript para asegurar una navegación fluida y facilitar el contacto o las reservas.

Ver Demo Código en GitHub

Proyecto Floristería

Sitio web para una floristería, diseñado para capturar la belleza de sus arreglos florales y facilitar la compra online. El proyecto fue creado con HTML, CSS y JavaScript, buscando un diseño elegante y una experiencia de usuario que transmita la delicadeza de sus productos.
